Applied Artificial Intelligence Methods

A comprehensive hands-on course covering the practical implementation of AI algorithms, including Fuzzy Systems, Clustering, Neural Networks, Computer Vision, and Generative Models using MATLAB.

Instructor: Karla Miriam Reyes Leiva - Ankit Gupta - Martin Cerny

Term: Summer Semester (LS)

Course Overview

This course bridges the gap between theoretical Artificial Intelligence and practical software implementation. Students will learn to build, train, and evaluate various AI models from scratch using Python. The curriculum progresses from foundational rule-based systems (Fuzzy Logic) to advanced deep learning architectures (Generative Autoencoders and Convolutional Computer Vision filters).

By the end of this course, students will be able to:

  • Design and implement Fuzzy Inference Systems (Mamdani/Sugeno).
  • Build clustering algorithms and classification models without relying on high-level wrapper libraries.
  • Understand the mathematical foundations of straightforward Neural Networks.
  • Apply foundational computer vision techniques, and GANS.

Prerequisites

  • Proficiency in Python programming (functions, matrix operations, conv2).
  • Foundational understanding of linear algebra and calculus.
  • Basic knowledge of Git/GitHub for assignment submissions.

Grading & Requirements

  • Theoretical Exams: Conducted via the LMS (Multiple attempts allowed).
  • Practical Exams: Hands-on coding tests and Written theorical test.
  • Final Project : - Students must select one dataset to work on GANS.
    • Code must be fully commented.
    • Required Stand-up Presentation

Schedule

Week Date Topic Materials
1 Fuzzy Sets & Inference Systems

Introduction to linguistic variables, knowledge bases, membership functions, t-norms, and Mamdani’s/Sugeno approximative inference methods (ANFIS).
2 Classification Methods

Introduction to classification algorithms, evaluating models, and practical GitHub workflows for class activities
3 Machine Learning & Clustering

Implementing the K-means algorithm, calculating euclidean distances, cluster analysis, and evaluating with Confusion Matrices.
4 Artificial Neural Networks & Perceptrons

Principle and structure of straightforward neural networks, linear separability, input/output layer configurations, and class activation maps
5 Computer Vision & Image Segmentation

Binary and multiregional thresholding, Otsu’s method, and edge detection using the Sobel operator (convolutional masks).
6 Generative Models

Exploring generative architectures with a specific focus on Autoencoders.
7 Final Project Presentations

Stand-up presentation of final implemented projects (K-means from scratch, Sobel edge detectors, Otsu’s method, or predefined MATLAB NNs).
